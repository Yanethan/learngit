learn git

1.创建一个版本库非常简单，首先，选择一个合适的地方，创建一个空目录：
$ mkdir learngit
$ cd learngit
$ pwd
/Users/michael/learngit
pwd命令用于显示当前目录。

2.通过 git init 命令将这个目录变成git可以管理的仓库：
$ git init
Initialized empty Git repository in /Users/michael/learngit/.git/
.git 目录为隐藏目录，用 ls -ah 命令可以看见。

3.利用 $ git add 与文件名称  命令可以将文件添加到仓库，
接下来用 git commit -m "zhongjiantianjaibeizhu"命令可以查看文件状态

4.git diff 可以用来查看文件的different

5.git log 用于查看修改文件的日志
$ git log --pretty=oneline，这个参数用于显示版本ID与近三次修改：最近一次
修改为append GPL，上一次修改是add distributed,最早一次修改是wrote a readme file

6.版本回退方式：
    （1）.git reset --hard HEAD^
    (2).git reset --hard commit_id 可以恢复到相应的版本号。 
    （3）.git reflog 用于记录每一次命令

7.撤销修改 git checkout -- file

8. rm + 文件名 删除文件

9.创建分支 git checkout -b + 分支名称，-b表示创建并切换，相当于git branch+git checkout
使用git branch 命令查看当前分支

10.git stash 命令用于暂时存放当前状态文件，
    用git stash list 命令可以查看当前以存放列表
    用git stash apply 恢复存放文件，stash文件不删除
    用git stash pop 恢复文件同时删除stash文件
    也可以多次git stash后，使用先用git stash list查看，git stash apply stash@{0}恢复

11.打标签 git tag + 标签名（为最近一次commit打标签）
    为以前的提交打标签需 commit_id,打标签格式 git tag +标签名  +commit_id
    git tag命令可查看标签名
    git show +tagname查看标签信息
    命令git tag <name>用于新建一个标签，默认为HEAD，也可以指定一个commit id；
    git tag -a <tagname> -m "blablabla..."可以指定标签信息；
    git tag -s <tagname> -m "blablabla..."可以用PGP签名标签；
    命令git tag可以查看所有标签。
    命令git push origin <tagname>可以推送一个本地标签；
    命令git push origin --tags可以推送全部未推送过的本地标签；
    命令git tag -d <tagname>可以删除一个本地标签；
    命令git push origin :refs/tags/<tagname>可以删除一个远程标签。
